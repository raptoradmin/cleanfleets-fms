Programming

It would be necessary to create a programming module for the application perhaps called Grants.

In the Grants module data entry forms would have to be create to collect all of the data related to the Grant Proposal, Replacement Vehicle, Replacement Dealers, and Replacement Engines.

Business logic would have to be written to link any one existing vehicle/engine to the intended replacement vehicle/engine and relate the replacements with the Dealers.

If it was desired to be able to scan and attached the necessary paperwork to the replacement vehicles such as Title, Registration Papers, Images, etc., it would be necessary to create business logic and file allocation space for these files.

It would be necessary to create business logic to compute the calculations to provide sum totals for the requested vehicles and cost totals.

It would be necessary to create the reports to output the data in either text file, PDF or print format.
